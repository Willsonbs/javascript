JavaScript Operadores

Aritméticos

Soma: 5 + 3 = 8

Subtração: 10 - 2 = 8

Multiplicação: 3 * 4 = 12

Divisão: 10 / 2 = 5

Resto da divisão: 10 % 3 = 1

Atribuição

Adição e atribuição: var x = 5; x += 3;

Subtração e atribuição: var y = 10; y -= 2;

Relacionais

Maior: 5 > 3 true

Menor: 5 < 3 false

Maior ou igual: 8 >= 8 true

Menor ou igual: 9 <= 7 false

Igualdade: 5 == 5 true

Diferente: 5!= 5 false

Negação

! negação

true or false

&& conjunção

true && true == true

true && false == false

false && true == false

false && false == false

|| disjunção

true || true == true

true || false == true

false || true == true

false || false == false
